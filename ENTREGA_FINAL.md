## Resumen del Proyecto
Este repositorio contiene el pipeline completo de MLOps para la predicción de Churn en TelcoVision.
Incluye:
- Versionado de datos con DVC y DagsHub.
- Pipeline reproducible (Data Prep -> Train -> Evaluate).
- Tracking de experimentos con MLflow.
- CI/CD con GitHub Actions.
- API de producción con FastAPI.
